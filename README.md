# 微信小程序

基于[taro react](https://taro-docs.jd.com/docs/)开发

## 安装依赖

```shell
pnpm i
```

### 启动

根目录下运行以下命令

```shell
pnpm dev:weapp
```

## 打包

```shell
pnpm prod:build:weapp
```
